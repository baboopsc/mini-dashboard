\# Mini-Dashboard Projesi 

Bu proje, Git ile iş birliği ve çakışma (conflict) yönetimi üzerine bir eğitim ödevi olarak hazırlanmıştır.



\##  Ekip Üyeleri

\- \*\*Öğrenci B (Modül Geliştirici):\*\* Okan Karahan 23253035 (@oklies37)



\##  Proje Yapısı

\- `main.py`: Uygulamanın ana giriş noktası. Modülleri birleştirir.

\- `weather.py`: Hava durumu bilgilerini sağlayan modül.

\- `news.py`: Güncel haber bilgilerini sağlayan modül.



\##  Öğrenilen Kazanımlar

\- GitHub üzerinde \*\*Collaborator\*\* yönetimi.

\- Farklı dosyalarda çalışarak modüler kod geliştirme.

\- \*\*Merge Conflict\*\* (Çakışma) oluşturma ve çözme süreci.

\- Git pull/push döngüsü ile ekip çalışması senkronizasyonu.
